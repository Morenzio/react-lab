# react-lib-prova

> Made with create-react-library

[![NPM](https://img.shields.io/npm/v/react-lib-prova.svg)](https://www.npmjs.com/package/react-lib-prova) [![JavaScript Style Guide](https://img.shields.io/badge/code_style-standard-brightgreen.svg)](https://standardjs.com)

## Install

```bash
npm install --save react-lib-prova
```

## Usage

```tsx
import React, { Component } from 'react'

import MyComponent from 'react-lib-prova'
import 'react-lib-prova/dist/index.css'

class Example extends Component {
  render() {
    return <MyComponent />
  }
}
```
